## 📖 Description

**holbertonschool-project-paris** est un site web statique sur le thème de **Paris**, développé en équipe dans le cadre du cursus Holberton School. Le projet vise à mettre en pratique les fondamentaux du développement web front-end : HTML sémantique, CSS mise en page, et travail collaboratif via Git.

---

## 🎯 Objectifs

- Maîtriser la structure HTML sémantique
- Appliquer des styles CSS avancés (flexbox, mise en page responsive)
- Collaborer efficacement via Git et GitHub (branches, pull requests)
- Produire un rendu visuel soigné sur une thématique culturelle

## ✅ Prérequis

- Un navigateur web moderne (Chrome, Firefox, Safari, Edge)
- Git installé (`git --version`)
- Aucun package supplémentaire requis — projet 100% statique

---

## 🚀 Installation

```bash
# Cloner le dépôt
git clone https://github.com/Rwanbt/holbertonschool-project-paris.git

# Se déplacer dans le répertoire
cd holbertonschool-project-paris
```

---

## ▶️ Utilisation

### Option 1 — Ouvrir directement dans le navigateur

```bash
# Linux
xdg-open index.html

# macOS
open index.html

# Windows
start index.html
```

### Option 2 — Serveur local (recommandé)

```bash
# Avec Python 3
python3 -m http.server 8080

# Puis ouvrir dans le navigateur :
# http://localhost:8080
```
